# Security Policy

## Supported Versions

Autogex is an active project. We apply security updates and bug fixes primarily to the latest major/minor release.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Autogex, please **do not** open a public issue.

Instead, please report it via [GitHub Security Advisories](https://github.com/Tugamer89/autogex/security/advisories/new) or send a direct email to the repository owner.

We take all security reports seriously and will strive to acknowledge your report within 48 hours. A fix will be provided as quickly as possible, followed by a patch release.
